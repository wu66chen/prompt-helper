## ✨核心功能亮点
- **全网跨端同步**：支持在 豆包、ChatGPT、DeepSeek、Kimi、QWen、Gemini 等各大主流 AI 平台呼出。由于数据存储在油猴底层，你在 ChatGPT 存的提示词，转头打开 DeepSeek 就能立刻无缝调用，彻底打通平台壁垒。
- **所见即所得的“红字变量”引擎**：抛弃繁琐的 {{变量}} 符号！用鼠标选中文本并点击 【标红选中文本】，即可生成醒目的红色变量区。支持随时框选 【标黑】 恢复普通状态，模板结构一目了然。
- **一键直拷**：急需使用提示词？只需在列表中点击提示词的文本区域，脚本会自动为你剥离所有复杂的底层变量逻辑，瞬间将一份纯净无格式的文本送入你的剪贴板，秒速发送！
- **深/浅色模式**：点击面板左上角的 ☀️/🌙 按钮即可切换深/浅色主题。
- **左上角拉伸视窗**：主面板左上角蓝色三角为拉伸窗口按钮，按住它向左上方自由拖拽，满足多种提示词数量的使用场景。
- **标签云与极速搜索过滤**：支持为单条提示词打上多个标签。列表上方内置了 🔍搜索过滤框，输入关键字瞬间在成百上千的标签和列表中找到你的目标。
- **双轨编辑架构**：
  - **临时编辑（高频调用）**：在独立窗口中临时将红字替换为当次所需的对话内容，点击复制即可使用，完全不污染原模板。
  - **修改（永久保存）**：对提示词模板本身进行长期维护。
- **导入/导出提示词**：点击“导出TXT”，脚本会利用零宽字符隐写术将红黑变量标记隐藏。你导出的 TXT 在肉眼看来是排版极度纯净的文本；但当你“导入TXT”时，脚本能重新识别出隐形标记，完美恢复所有的红字与标签，且自带智能去重功能防冲突。

## 🧭快捷操作指北
- 展开面板：点击网页右下角 💡 悬浮蓝球。
- 放大窗口：鼠标对准面板左上角的蓝色斜角区，按住拖拽。
- 快捷复制：鼠标移动到任意提示词文本上，背景变灰时，点击左键即可直接复制。
- 主题切换：点击面板顶部的 🌙 / ☀️ 图标。

## ⚠️数据安全必读
- 数据储存于你浏览器的 **油猴 (Tampermonkey) 本地环境** 中，绝无任何云端服务器窃取您的隐私。
- 卸载油猴插件、重装浏览器、或使用管家清理了“扩展程序数据”，均会导致提示词消失。
- 强烈建议您定期点击 【导出TXT】 进行备份。
- 关于导出的TXT文件：请不要在电脑记事本里随意去修改导出的文本，否则可能会破坏隐藏在文字旁边的“零宽占位符”，导致再次导入时无法精准还原红色文字！若用于异地恢复，原样传输即可。
